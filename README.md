# eigen_examples
## About
Eigen3の使い方を例とともに記す．

## Examples
- 2d_transform : 2次元空間における座標変換に関わるもの


## Build & Execute
#### Build
```bash
$ source compile.bash
```

#### Execute
`eigenrun`コマンドが使えるようになっている．  
TAB押下により，実行プログラム名の補完が行われる．
```bash
$ eigenrun <TAB>
```
